# Agri-Assist Frontend

Frontend for Agri-Assist, a web app that helps chili farmers identify likely plant diseases from visible symptoms.

## Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- shadcn-style UI primitives
- Vitest
- Playwright

## Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Scripts

```bash
npm run dev
npm run build
npm run start
npm run typecheck
npm run test
npm run test:e2e
```

## Current Scope

- Landing page
- Symptom selection flow
- Mock weighted diagnosis results
- Recommendation pillars
- Feedback UI
